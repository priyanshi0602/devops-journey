# Day 43 - Docker Compose Multi-Service Application

## Objective

Deploy multiple services using Docker Compose.

## Topics Covered

* Docker Compose
* Multi-Service Applications
* PostgreSQL
* Named Volumes
* Environment Variables
* Compose Configuration

## Commands Used

docker compose up -d

docker compose ps

docker compose logs

docker compose config

docker compose down

## Key Learnings

* Docker Compose can manage multiple services together.
* PostgreSQL data is persisted using a named volume.
* Environment variables configure database settings.
* Compose simplifies application deployment.

## Outcome

Successfully deployed an Nginx web server and PostgreSQL database using a single Compose file.

# Day 44 - Advanced Docker Compose

## Objective

Learn production-ready Docker Compose features.

## Topics Covered

* depends_on
* restart policies
* .env
* custom networks
* named volumes

## Commands

docker compose up -d

docker compose ps

docker network ls

docker network inspect

docker compose down

## Key Learnings

* Services can depend on each other.
* Environment variables improve security.
* Custom networks simplify communication.
* Restart policies improve reliability.

## Outcome

Successfully deployed a production-style multi-service Docker Compose application.

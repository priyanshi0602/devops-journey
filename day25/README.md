# Day 25 - Flask with Docker Compose

## Objective

Learn how to run a Flask application using Docker Compose.

## Project Structure

* app.py
* requirements.txt
* Dockerfile
* docker-compose.yml
* README.md

## What I Learned

* Docker Compose workflow
* Building images through Compose
* Running containers using Compose
* Port mapping
* Managing applications with a YAML configuration

## Commands Used

```bash
docker compose up -d

docker compose ps

docker ps

docker compose down
```

## Result

Successfully deployed a Flask application using Docker Compose.

Application URL:

http://localhost:5001

Output:

Hello from Day 25 Flask + Docker Compose!

## Key Takeaway

Docker Compose simplifies application deployment by defining services and configuration in a single file.

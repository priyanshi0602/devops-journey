# Day 42 - Docker Compose Basics

## Objective

Learn how to manage containers using Docker Compose.

## Topics Covered

* Docker Compose
* compose.yaml
* Services
* docker compose up
* docker compose down
* docker compose logs

## Commands Used

docker compose up -d

docker compose ps

docker compose logs

docker compose down

## Key Learnings

* Docker Compose manages multiple containers from one file.
* compose.yaml defines application services.
* One command can start or stop an entire application stack.
* Docker Compose simplifies local development.

## Outcome

Successfully deployed an Nginx container using Docker Compose and managed it using Compose commands.
